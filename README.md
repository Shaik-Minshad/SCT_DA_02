# Task 02: Data Cleaning and Preparation

## Objective
Load a dataset using Pandas, identify and handle missing values, remove duplicate rows, convert data types, and export the cleaned dataset.

## Dataset
Titanic Dataset

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Steps Performed
1. Loaded the dataset.
2. Identified missing values.
3. Filled missing values in Age and Embarked columns.
4. Removed the Cabin column due to excessive missing values.
5. Checked and removed duplicate rows.
6. Converted Sex, Embarked, and Pclass columns to categorical data types.
7. Exported the cleaned dataset to a new CSV file.

## Output
cleaned_titanic.csv
